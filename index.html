My Baseball Dice Game

Written by Caesar A.

This is a simple little baseball dice game I came up with a while ago when I was bored. At the time, I did make a few different versions, but this is by far the best one. It uses two six-sided dice and a special result list to simulate batting outcomes. The way the game works is that players roll the dice and use their sum to reference the result list.

The Result List

  • 2 - Ground Out
  • 3 - Triple
  • 4 - Single (Double if 2/2 is rolled)
  • 5 - Fly Out
  • 6 - Strike Out
  • 7 - Pop Out
  • 8 - Strike Out
  • 9 - Fly Out
  • 10 - Single (Double if 5/5 is rolled)
  • 11 - Home Run
  • 12 - Ground Out

Because this game uses a highly simplified version of the standard baseball rules, only things such as base runner positions, outs, scored runs, and innings must be kept track of by the player. Things such as strikes, balls, errors, and other stuff are completely ignored. Below are some statistics, which were calculated from the aforementioned result list.

In Game Statistics

  • Hits - 6
  • Outs - 15
  • At-Bats - 21
  • Batting Average - .286

I like this game quite a lot because the rules are easy to remember and it is very fun to play.
